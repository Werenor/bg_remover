# bg_remover

一个用于批量去除 PNG 背景并清理边缘残色的简单桌面工具。

## 功能

- 选择输入目录和输出目录
- 批量处理 PNG 图片
- 根据指定背景色移除背景
- 将背景区域转换为透明
- 清理主体边缘残留的背景染色
- 提供简单参数调节界面

## 适用场景

这个工具适合处理这类图片：

- 背景为纯色或近似纯色
- 需要将背景变为透明
- 边缘存在轻微背景残色
- 例如游戏素材、序列帧、导出贴图等

## 运行环境

- Python 3.14
- Pillow
- tkinter

其中 `tkinter` 为 Python 标准库自带，通常无需单独安装。

## 安装依赖

```bash
pip install pillow


## 运行方式

在项目目录执行：

```bash
python main.py
```

## 使用方法

1. 启动程序
2. 选择输入目录
3. 选择输出目录
4. 设置背景色 RGB
5. 调整“背景匹配容差”
6. 调整“边缘去残色强度”
7. 点击“开始处理”

## 参数说明

### 背景色 RGB

用于指定需要移除的背景参考颜色。

默认值：

* R = 255
* G = 0
* B = 255

### 背景匹配容差

控制哪些像素会被视为背景。

* 数值越大，越容易删除接近背景色的像素
* 数值过大时，可能误伤主体边缘

建议范围：

* 60 ~ 100

### 边缘去残色强度

用于减轻主体边缘残留的背景染色或杂色。

* 数值越大，边缘残色清理越明显
* 数值过大时，边缘可能发灰或失真

建议范围：

* 0.7 ~ 1.2

## 输出说明

* 仅处理输入目录中的 PNG 文件
* 输出文件格式仍为 PNG
* 输出到所选输出目录
* 原文件不会被直接覆盖

## 已知限制

* 更适合背景较单一的图片
* 对复杂背景、多色背景或强渐变背景效果有限
* 如果原图边缘已经与背景发生明显混色，仍可能需要进一步微调参数

## 项目说明

这个工具最初用于处理带有纯色背景的 Unity 导出 PNG 序列帧，后整理为可复用的小工具。
